# Chirp

A social media platform.

## Deployment

To run our project go the src/Chirp.Razor folder:

```bash
  cd src/Chirp.Razor
```

Now to start the program run:

```bash
  dotnet run
```

Now you can go to http://localhost:5273 and see the website
